# Void Web App

React + Vite + Tailwind CSS

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

## Stack

- **React 19** — UI
- **Vite 8** — dev server & build
- **Tailwind CSS 3** — utility-first CSS
